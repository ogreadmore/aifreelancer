JWT Decoder

Decode JWT header and payload segments locally, pretty-print the JSON, and inspect time-based claims in both local and UTC formats.

Signature warning

This tool decodes the token locally, but it does not verify signatures. Use it to inspect structure and claims, not to trust the token by itself.

Good fit

  • Inspecting access tokens and ID tokens
  • Reviewing exp, nbf, iat, and auth_time claims
  • Checking tokens without calling a backend

Everything stays in the browser. Nothing is uploaded anywhere.
