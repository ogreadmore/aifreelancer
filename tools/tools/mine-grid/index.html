How to play

Classic logic, polished for quick browser sessions.

Rules

  • Open safe tiles to reveal the adjacent mine counts.
  • Use right-click or flag mode to mark likely mines.
  • Clear every safe tile to set a local best time.

Keyboard

  • 1 2 3 to switch presets
  • F toggles flag mode
  • R restarts the current board

Local records

Flow tip: the first reveal is guaranteed safe and opens a small pocket. Start near center if you want the strongest opening.
