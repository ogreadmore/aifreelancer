UTM Builder

Build campaign URLs locally in the browser by combining a base URL with UTM source, medium, campaign, term, and content fields. Values are sanitized, encoded, and previewed instantly.

What it does

  • Preserves existing query strings and hash fragments on your base URL
  • Encodes all UTM values safely with the browser URL APIs
  • Keeps everything local with no dependencies or network calls
