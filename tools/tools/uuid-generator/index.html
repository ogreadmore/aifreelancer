Quick guide

What the format means

Version

4

The third group starts with `4` to indicate a random UUID v4.

Variant

8, 9, A, or B

The first character of the fourth group follows the RFC 4122 variant rule.

Length

36

The dashed UUID form uses 32 hex digits plus 4 hyphens.

Use cases

Great for database IDs, request tracing, test fixtures, and temporary tokens.
